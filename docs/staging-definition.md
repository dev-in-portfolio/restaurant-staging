# What is the Staging Stage?

The **Ready for Polish** (or **Survivors' Lounge**) stage is the second phase of our website delivery pipeline.

## Purpose
Staging bridges the gap between raw concept prototypes in the Thunderdome and presentation-ready showpieces in the Showcase. It serves as a quality gate where layout corrections, text auditing, address/phone checkups, and image sizing are completed.

## Staging Requirements
A website folder in staging contains:
1. Exact static pages for the restaurant (6-page standard).
2. The `restaurant.json` metadata record indicating that staging reviews are active.
3. The floating current-site comparison widget.
4. Clean assets (no rejected images or generator garbage).
